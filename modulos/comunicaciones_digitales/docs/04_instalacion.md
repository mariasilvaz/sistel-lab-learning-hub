# Instalacion

Pendiente hasta que se desarrolle el primer tema y se sepa con certeza que funciones de MATLAB y que toolboxes se usaron efectivamente (ver estandar_codigo.md, seccion "Manejo de dependencias").

Cuando se complete, debe incluir: version minima de MATLAB, toolboxes requeridos (Communications Toolbox y Signal Processing Toolbox son los previstos por la licencia campus disponible, pero deben confirmarse), y pasos para dejar el modulo funcionando (ej. agregar carpetas al path).
