# Guia de desarrollador

## Convenciones ya definidas para este modulo

- Arquitectura de fuente intercambiable (`funciones/fuente/`) y cuantizacion compartida (`funciones/cuantizacion/`): ver docs/03_arquitectura.md antes de agregar una funcion nueva relacionada con generacion de datos o cuantizacion, para no duplicar lo que ya existe ahi.
- 21 temas independientes en `practicas/`: al agregar un tema nuevo que no encaje claramente en los 21 ya definidos, discutirlo primero en un Issue antes de crear una carpeta 22 en adelante.
