# Canales continuos de entrada discreta y de forma de onda

**Estado:** Pendiente de desarrollo -- sin script de sistema ni guia todavia.

## Alcance de este tema

Temario 4.6-4.7.

## Como empezar a desarrollar este tema

1. Abre (o reclama) el Issue correspondiente a este tema usando la plantilla nuevo_modulo.md o tarea_desarrollo.md.
2. Sigue docs/estandares/estandar_codigo.md, estandar_documentacion.md y estandar_pedagogico.md.
3. Parte de las plantillas en docs/plantillas/:
   - plantilla_script_sistema.m -> el script de sistema de este tema.
   - plantilla_guia.md -> la guia de practica de este tema.
4. Si este tema necesita una funcion nueva y reutilizable, agregala en ../../funciones/ (o en la subcarpeta correspondiente, ej. fuente/ o cuantizacion/) siguiendo plantilla_funcion.m, no la escribas solo dentro de este script.

Al completarse, este archivo se reemplaza por el contenido real del tema y su estado se actualiza en el README.md del modulo.
